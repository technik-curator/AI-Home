Small runnable experiments demonstrating ideas from AI-Home.

## 2) Structure
Each demo lives in its own folder:
- `demos/<demo-name>/` — code + README + assets

## 3) How to run
Each demo folder must contain:
- `README.md` with steps
- (optional) `requirements.txt` / `package.json`

## 4) Publish rules (IMPORTANT)
- ✅ Demos must run without secrets (use env vars / placeholders).
- ❌ Never commit `.env`, tokens, private endpoints.
- ✅ Add `.gitignore` inside demo if needed (logs, cache, local data).
